# Non-Tripod Landing Page Design System

## Color Palette
- **Primary:** `#E30613` (Perfa-Pik Red)
- **Secondary:** `#000000` (Pitch Black)
- **Accents:** `#1D1D1F` (Space Grey / Apple Black)
- **Text:** `#F5F5F7` (Off-white) / `#86868B` (Grey Text)
- **Background:** `#000000` (Main), `#161617` (Section alternations)

## Typography
- **Heading:** `Outfit`, sans-serif (Bold, weighted)
- **Body:** `Inter`, sans-serif

## Components
1. **Nav:** Sticky, glassmorphism (`backdrop-filter: blur(20px)`), minimal logo.
2. **Hero:** Full-screen emphasis, large centered text, scroll-down hint.
3. **Feature Grid:** Bento-box style or large scroll sections.
4. **CTA:** Magnetic button effect or high-contrast red buttons.

## Animations
- **Intersection Observer:** For "Fade up" and "Blur in" effects on scroll.
- **Parallax:** Subtle on hero image.
